Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd	: fer í $HOME skránna
	cd ~	: fer upp skref í skrá treið
	ls	: sýnir öll upplýsinga um skjölinn eða skránna
	pwd	: sýnir hvar þú ert staddur
	mkdir	: býr til skrá
2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone	: býr til copy af local repository
	git log		: 
	git status	: listar skránna sem þu hefur breytt og sýnir þá sem þu átt eftir að bæta eða fremja
	git diff	: sýnir öll sameina átök
	git checkout	: skiftir frá einni grein í annar

4. 	Hvað er útgáfustýring (e.version control)? 
	eru mörg hugbúnaði verkfæri sem hjálpar sum hugbúnað lið til að stýra og breyta í source code.
5.	Hverjir eru helstu kostir við að nota GIT?
	Getur unnið í verkefni yfir netið með öðru fólki.
6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	eins oft og hægt er, því ef maður geri villa getur þú farið til baka.
7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	Working directory: er stakur athuga af ein útgáfa

	Staging area: The staging area is a file, generally contained in your Git directory, 
	that stores information about what will go into your next commit.

	Repository: The Git directory is where Git stores the metadata and object database for your project.
8.	Hvenær er sniðugt að nota greinar (branches)? 
	þegar það er unnið í verkefnið þegar það er fleiri en ein.
